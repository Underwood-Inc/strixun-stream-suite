## Audit Report Structure

All audit reports follow this structure:

1. **Executive Summary** - High-level findings
2. **Scope** - What was audited
3. **Findings** - Detailed findings organized by category
4. **Recommendations** - Actionable recommendations
5. **Status** - Current status of recommendations
