# E2E Helpers - End-to-End Testing Utilities

Shared utilities for end-to-end testing across all Strixun Stream Suite projects.

## Overview

This package provides helper functions and utilities for E2E testing, including email interception, test fixtures, and common testing patterns.

## Features

- ✓ **Email Interception** - Intercept and verify email OTP codes in tests
- ✓ **Test Fixtures** - Reusable test data and fixtures
- ✓ **Helper Functions** - Common testing utilities
- ✓ **TypeScript** - Fully typed

## Usage

```typescript
import { interceptEmail, getTestFixtures } from '@strixun/e2e-helpers';
```

## Installation

This package is part of the pnpm workspace and is automatically available to all packages.

## Development

No build step required - this is a TypeScript-only package.
