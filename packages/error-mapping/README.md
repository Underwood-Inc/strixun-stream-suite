# Error Mapping - Error Mapping Utilities

Error mapping utilities with error legend for consistent error handling across applications.

## Overview

This package provides error mapping functionality to ensure consistent error messages and codes across all services.

## Features

- ✓ **Error Legend** - Centralized error code definitions
- ✓ **Error Mapping** - Map error codes to user-friendly messages
- ✓ **TypeScript** - Fully typed

## Usage

```typescript
import { getErrorCode, getErrorMessage } from '@strixun/error-mapping';
```

## Installation

This package is part of the pnpm workspace and is automatically available to all packages.

## Development

No build step required - this is a TypeScript-only package.
