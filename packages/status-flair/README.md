# Status Flair - Status Badge Component

Status badge/flair component for displaying status information.

## Overview

A simple, reusable component for displaying status badges with different styles and colors.

## Features

- ✓ **Svelte Component** - Native Svelte component
- ✓ **Multiple Styles** - Different badge styles
- ✓ **TypeScript** - Fully typed

## Usage

```svelte
<script>
  import { StatusFlair } from '@strixun/status-flair';
</script>

<StatusFlair status="active" />
```

## Installation

This package is part of the pnpm workspace and is automatically available to all packages.

## Development

```bash
cd packages/status-flair
pnpm install
```
