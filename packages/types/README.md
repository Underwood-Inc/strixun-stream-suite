# Types - Shared TypeScript Types

Shared TypeScript type definitions used across all Strixun Stream Suite projects.

## Overview

This package provides common type definitions to ensure type consistency across the monorepo.

## Usage

```typescript
import type { SomeType } from '@strixun/types';
```

## Features

- ✓ **Shared Types** - Common type definitions
- ✓ **Type Safety** - Ensures consistency across projects
- ✓ **Zero Dependencies** - Pure TypeScript types

## Installation

This package is part of the pnpm workspace and is automatically available to all packages.

## Development

No build step required - this is a TypeScript-only package.
