# Ad Carousel - Svelte Component

Ad carousel component for Svelte with Twitch integration support.

## Overview

A flexible carousel component for displaying ads, with built-in support for Twitch ads and customizable styling.

## Features

- ✓ **Svelte Component** - Native Svelte component
- ✓ **Twitch Integration** - Built-in Twitch ad support
- ✓ **Customizable** - Flexible styling and configuration
- ✓ **TypeScript** - Fully typed

## Usage

```svelte
<script>
  import { AdCarousel } from '@strixun/ad-carousel';
</script>

<AdCarousel ads={ads} />
```

## Installation

This package is part of the pnpm workspace and is automatically available to all packages.

## Development

```bash
cd packages/ad-carousel
pnpm install
```
